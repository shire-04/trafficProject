## 3. 核心推理路径与真实示例

### 3.1 Action -[:NEXT_STEP]-> Action (1,405条)
**用途**: 任务分解,将高层任务分解为具体执行步骤

**Cypher查询**:
```cypher
MATCH (a1:Action)-[:NEXT_STEP]->(a2:Action)
RETURN a1.name, a2.name
LIMIT 3
```

**真实示例**:
- `组织成员单位对遇险人员进行紧急援救` → `破拆救援被困人员`
- `组织成员单位对遇险人员进行紧急援救` → `转运受伤人员`
- `组织成员单位对遇险人员进行紧急援救` → `救援被困人员`

---

### 3.2 Action -[:REQUIRES]-> Resource (970条)
**用途**: 资源调度,定义动作所需的人力、车辆、设施资源

**Cypher查询**:
```cypher
MATCH (a:Action)-[:REQUIRES]->(r:Resource)
RETURN a.name, r.name
LIMIT 3
```

**真实示例**:
- `执行交通管制` → `交警`
- `事故勘查定责` → `交警`
- `车辆分流引导` → `交警`

---

### 3.3 Consequence -[:CONSISTS_OF]-> Action (457条)
**用途**: 后果包含动作,定义处理特定后果所需的具体措施

**Cypher查询**:
```cypher
MATCH (c:Consequence)-[:CONSISTS_OF]->(a:Action)
RETURN c.name, a.name
LIMIT 3
```

**真实示例**:
- `苯酚与液化天然气泄漏` → `处置危化品泄漏`
- `苯酚与液化天然气泄漏` → `检测危化品泄漏对空气/水质的污染`
- `苯酚与液化天然气泄漏` → `稀释危化品泄漏物`

---

### 3.4 Event -[:TRIGGERS]-> Action (455条)
**用途**: 事件触发响应动作,定义特定事件或等级对应的处置措施

**Cypher查询**:
```cypher
MATCH (e:Event)-[:TRIGGERS]->(a:Action)
RETURN e.name, a.name
LIMIT 3
```

**真实示例**:
- `公路交通突发事件等级` → `市公路交通应急指挥部启动应急响应`
- `公路交通突发事件等级` → `指挥各区县（开发区）公路交通应急指挥机构做好处置工作`
- `公路交通突发事件等级` → `汇总、整理信息，向市公路交通应急指挥部报告，并通报各成员单位`

---

### 3.5 Event -[:LEADS_TO]-> Consequence (140条)
**用途**: 因果推理,事件导致的直接后果

**Cypher查询**:
```cypher
MATCH (e:Event)-[:LEADS_TO]->(c:Consequence)
RETURN e.name, c.name
LIMIT 3
```

**真实示例**:
- `液化天然气车追尾苯酚车` → `苯酚与液化天然气泄漏`
- `苯酚与液化天然气泄漏` → `2人轻伤`
- `普通货车追尾苯槽罐车` → `苯少量泄漏`

---

### 3.6 Event -[:REQUIRES]-> Action (103条)
**用途**: 事件直接触发的应对动作

**Cypher查询**:
```cypher
MATCH (e:Event)-[:REQUIRES]->(a:Action)
RETURN e.name, a.name
LIMIT 3
```

**真实示例**:
- `特别重大事件（I级）` → `启动应急预案`
- `特别重大事件（I级）` → `组织成员单位对遇险人员进行紧急援救`
- `特别重大事件（I级）` → `协调部队参加抢险救灾`

---

### 3.7 Action -[:MITIGATES]-> Consequence (50条)
**用途**: 动作缓解后果,定义哪些措施可以减轻特定后果

**Cypher查询**:
```cypher
MATCH (a:Action)-[:MITIGATES]->(c:Consequence)
RETURN a.name, c.name
LIMIT 3
```

**真实示例**:
- `依据《中华人民共和国道路交通安全法》第三十九条实施隧道临时交通管制+按照《国家交通应急预案》第三章调派危化品消防队+分批疏散滞留司乘+同步开展环境监测和排烟` → `隧道内汽柴油大量泄漏并产生有毒烟雾`
- `依《中华人民共和国道路交通安全法》第六十八条组织交通分流+按《国家交通应急预案》第四章启动危险化学品专项响应+调集桥梁检测与防冻队伍+实施高架全线封闭除冰` → `高架桥多车挤压变形伴随危化品泄漏`
